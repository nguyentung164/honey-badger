## 📝 Changelog / 変更履歴 / Thay đổi

### English (en)

- Changed GitHub repository owner in electron-builder.ts.
- Updated author information in package.json and README.md.
- Removed unused files: EVM_Tool.txt, getRefreshToken.md, and unused-translation-keys.txt.
- Updated license year in LICENSE file.
- Added new IPC constants in constants.ts for PR manager functionality.
- Enhanced PR handling in GitHub integration with optional review submissions.
- Improved window management for PR manager in window.ts.
- Refactored settings dialogs to use accordion components for better UI organization.
- Updated version number in package.json to 1.0.1.
